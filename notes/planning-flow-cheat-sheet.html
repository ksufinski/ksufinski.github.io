Planning flow cheat sheet

As is

AKA problem statement, description of current problem.

To be

Final goal (result) that should be instead of problem. SMART form fits best.

Implementation steps

How to split changes into milestones, milestones to small, atomic actions. What should be changed first.

Bottle necks

Predict risks, ways to deal with them.

Verification question

Specific question to verify that goal (milestone scope) was achieved.
