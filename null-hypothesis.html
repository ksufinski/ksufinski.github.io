Interactive Null Hypothesis Testing Visualization

Alpha (α = 0.05): The probability of a Type I error, i.e., rejecting the null hypothesis when it is actually true.

Power (1−β = 0.80): The probability of correctly rejecting the null hypothesis when the alternative hypothesis is true (i.e., avoiding a Type II error).

Effect Size (δ = 0.20): The standardized difference in means you aim to detect, expressed in units of the population standard deviation.
